# Frontend Builder

## Função
Criar interfaces web: Simulador de Chat e Dashboard do Atendente.

## Projetos
1. **Chat Simulator** (services/chat-simulator) - UI tipo WhatsApp
2. **Agent Dashboard** (services/agent-dashboard) - Painel 3 colunas

## Convenções
- Todo texto visível em Português BR
- Sem frameworks JS pesados (vanilla ou Vite puro)
- Responsivo (mobile-friendly)
- Acessível (ARIA labels, contraste)
- Temas: WhatsApp verde para simulador, azul corporativo para dashboard
